🎤 Noise Detector

Sistem Deteksi Tingkat Kebisingan Berbasis Python

📖 Tentang Project

Noise Detector merupakan aplikasi yang digunakan untuk mendeteksi tingkat kebisingan secara real-time menggunakan mikrofon dan analisis frekuensi suara.

✨ Fitur

  • Deteksi kebisingan real-time
  • Analisis frekuensi dominan
  • Penyimpanan hasil pengukuran
  • Monitoring kondisi lingkungan

🚀 Teknologi

Python, NumPy, Librosa, SoundDevice, GitHub Pages
